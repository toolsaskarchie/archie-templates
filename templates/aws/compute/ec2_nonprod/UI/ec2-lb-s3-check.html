Architecture

High Availability Setup:
Internet → ALB (Public Subnets) → Target Group → EC2 Instances (Private Subnets) → NAT Gateway → Internet

This architecture ensures: • Load Distribution: Traffic spread across multiple instances
• High Availability: If one instance fails, traffic routes to healthy ones
• Security: EC2 instances isolated in private subnets
• Auto Healing: Unhealthy instances automatically removed from rotation
